# Responsive Multi-Section Layout

A fully responsive multi-section homepage built mobile-first using HTML5, CSS3 (Flexbox + Grid), and vanilla JavaScript.

## Sections

- Header / Sticky Navbar (with hamburger menu on mobile)
- Hero Section
- Features / Services (CSS Grid — 1 → 2 → 4 columns)
- About Preview (Flexbox — stacked on mobile, side-by-side on desktop)
- Portfolio Preview (CSS Grid — 1 → 2 → 3 columns)
- Contact CTA
- Footer

## How to run

Just open `index.html` in your browser — no server needed.

## Technologies

- HTML5 (semantic tags throughout)
- CSS3 — Flexbox, Grid, custom properties, media queries
- JavaScript — hamburger toggle, sticky navbar shadow on scroll

## Responsive breakpoints

- Mobile: < 768px — 1 column, vertical nav
- Tablet: ≥ 768px — 2 columns, horizontal nav
- Desktop: ≥ 1024px — 3–4 columns, full layout
